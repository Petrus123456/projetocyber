# Explicação da Arquitetura

## External Network
Simula usuários e ameaças externas.

## DMZ
Contém a aplicação DVWA exposta para acesso web.

## Internal Network
Rede interna isolada contendo o banco de dados.

## Firewall
O firewall utiliza iptables para controlar o tráfego entre redes.

## Objetivo
Demonstrar conceitos de:
- segmentação de rede
- defesa em profundidade
- isolamento de serviços
- segurança ofensiva e defensiva
